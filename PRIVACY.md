# Privacy Policy

Memory Color Game is a student project created as part of the course Apputvikling 2.

## What data we collect
We collect the minimum amount of data required to identify a user in the system:
- A system-generated user ID
- An optional display name
- Timestamps for when the user accepted the Terms of Service and Privacy Policy

No email addresses, passwords, or other personal identifiers are collected.

## Why we collect this data
The data is used only to:
- Identify players in the game
- Associate game sessions with users
- Handle account creation and deletion

## Data storage
In the current version of the project, all data is stored in memory only and is deleted when the server is restarted.
No data is shared with third parties.

## User rights
Users have the right to:
- Access their data
- Delete their account and associated personal data

Deleting an account removes the user from the system.

## Consent
Users must actively accept this Privacy Policy before an account can be created.
